# Lab_1_OS
# Проект: Создание процессов

## Структура проекта

### 1. employee.h
Заголовочный файл с определением структуры данных:
```c
struct employee
{
    int num;                // идентификационный номер сотрудника
    char name[10];          // имя сотрудника
    double hours;           // количество отработанных часов
};
```

### 2. Creator (creator.c)
Утилита для создания бинарного файла с записями о сотрудниках.

**Аргументы командной строки:**
```
Creator <имя_файла> <количество_записей>
```

**Функциональность:**
- Создает бинарный файл с указанным именем
- Запрашивает ввод данных о сотрудниках с консоли
- Сохраняет данные в структурированном формате

### 3. Reporter (reporter.c)
Утилита для генерации текстового отчета на основе бинарного файла.

**Аргументы командной строки:**
```
Reporter <исходный_файл> <файл_отчета> <оплата_за_час>
```

**Функциональность:**
- Читает данные из бинарного файла
- Рассчитывает зарплату: `зарплата = часы × оплата_за_час`
- Создает текстовый отчет в указанный файл
- Содержит заголовок и таблицу с данными сотрудников


**Функциональность:**
1. Запрашивает имя бинарного файла и количество записей
2. Запускает Creator с переданными параметрами
3. Ожидает завершения Creator
4. Выводит содержимое созданного бинарного файла
5. Запрашивает имя файла отчета и оплату за час
6. Запускает Reporter с переданными параметрами
7. Ожидает завершения Reporter
8. Выводит отчет на консоль

## Использование

1. **Скомпилируйте все программы**
2. **Запустите основную программу:**

3. **Следуйте инструкциям в консоли**
